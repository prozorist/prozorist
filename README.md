# Prozorist

## For Developers

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Git](https://git-scm.com/)

### Optional

- [Visual Studio Code](https://code.visualstudio.com/)

### Cloning Git Repo

```
git clone https://github.com/prozorist/prozorist
cd prozorist
```

### Installing Dependencies

```
npm install
```

### Building WebSite

```
npm run build
```

### Output

```
web/index.html
```
